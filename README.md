# TWITTER


## Objetivo

Neste desafio, vamos replicar o feed de notícias do Twitter. Para tanto, seguiremos um fluxo de versões.

![giphy] (https://media.giphy.com/media/3ov9jP4RIGQCUQOScg/giphy.gif)


## Checklist
